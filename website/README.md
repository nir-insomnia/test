# 🎨 תבנית אתר אינטרנט אישי - עיצוב ניאו-ברוטליסטי

## 📋 תיאור הפרויקט

תבנית אתר אישי מודרנית ומרשימה עם עיצוב **ניאו-ברוטליסטי** - שילוב של צבעים נועזים, צורות גאומטריות, גבולות שחורים עבים, וטיפוגרפיה בולטת. האתר כולל 4 דפים נפרדים עם אנימציות מתקדמות ופונקציונליות אינטראקטיבית.

### 🎯 מה חדש בגרסה זו?
- ✨ עיצוב חדשני וייחודי שבולט מהמון
- 🎨 פלטת צבעים עשירה וחיה
- 🎭 אנימציות מתקדמות ומרשימות
- 📄 דפים נפרדים במקום טאבים
- 🎮 משחק משופר עם מעקב אחר שיא וניצחונות
- 📱 רספונסיבי מלא - נראה מעולה בכל מכשיר

## 📁 מבנה הקבצים

```
├── index.html      - דף הבית
├── about.html      - דף אודות
├── blog.html       - דף בלוג

└── README.md       - המדריך הזה
```

## 🎨 עקרונות העיצוב

### סגנון ניאו-ברוטליסטי
- גבולות שחורים עבים (4px)
- צללים קשים (Hard shadows)
- צבעים עזים - אדום, תכלת, צהוב, סגול
- טיפוגרפיה נועזת - Rubik ו-Heebo
- צורות גאומטריות מודגשות

### פלטת צבעים
```css
--primary-color: #FF6B6B     (אדום)
--secondary-color: #4ECDC4   (תכלת)
--accent-yellow: #FFD93D     (צהוב)
--accent-purple: #A78BFA     (סגול)
--accent-green: #6BCF7F      (ירוק)
```

## 🏗️ מבנה האתר

### דף הבית
- Hero Section עם צורות מרחפות
- תפריט ניווט צף
- כרטיס היכרות
